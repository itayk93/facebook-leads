# Google Facebook Leads Scraper

סקרייפר ללידים מקבוצות פייסבוק דרך גוגל.

## התקנה
```bash
npm install
npx playwright install
```

## הרצה
```bash
node scraper.js
```

## מה הסקרייפר עושה
1. מחפש בגוגל פוסטים מקבוצות פייסבוק
2. מסנן לפי מילות מפתח (מפתחים, פרילנסרים, וכו')
3. שומר תוצאות ל- leads.json

## שדרוגים אפשריים
- הוספת פילטור לפי זמן
- מעבר בין עמודים
- שליפת תוכן מלא מהפוסטים
